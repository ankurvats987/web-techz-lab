# Week 10 React Monorepo

This folder contains three separate React applications in a single npm workspace monorepo.

## Projects

- `packages/exercise-1-form-validation`
- `packages/exercise-2-dynamic-list`
- `packages/exercise-3-api-fetch`

All projects share a single root `node_modules` and one root `package-lock.json`.

## Install

Run from `Week 10`:

```bash
npm install
```

## Run

```bash
npm run dev:ex1
npm run dev:ex2
npm run dev:ex3
```

## Build

```bash
npm run build
```

Or per exercise:

```bash
npm run build:ex1
npm run build:ex2
npm run build:ex3
```
